# 🎯 Overview

Car Rental is a modern full-stack car rental platform that allows users to browse available vehicles, view detailed car information, book cars with ease, and manage their own listings. The platform provides a secure authentication system, a smooth booking experience, and an intuitive responsive interface for both customers and car owners.

---

# ✨ Features

- 🔐 **Secure Authentication** — User authentication with Email/Password and Google Sign-In using BetterAuth with JWT-based authorization.
- 🚘 **Car Management** — Users can add, update, and delete their own car listings through a personalized dashboard.
- 📅 **Easy Booking System** — Book available cars with rental dates, special requests, and automatic total cost calculation.
- 📋 **My Bookings** — View, manage, and track all personal car bookings from one place.
- 🚗 **My Added Cars** — Car owners can manage all of their listed vehicles with full edit and delete functionality.
- 🔒 **Protected Routes** — Private pages are secured using JWT authentication and HTTP-only cookies.
- 📱 **Responsive Design** — Fully optimized for desktop, tablet, and mobile devices with a clean, modern UI.
